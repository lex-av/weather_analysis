Weather Analysis Project
========================
Финальный проект

Задание:
Необходимо написать консольную утилиту для многопоточной обработки данных,
аккумулирования результатов через API из Интернета и их дальнейшего представления
на графиках.

Входные данные:
Входные данные находятся в каталоге /data в данном репозитории в упакованном виде в
файле hotels.zip.
Внутри данного архива расположено несколько файлов в формате .CSV, содержащих множество
строк с информацией по отелям. В том числе, среди информации вы найдете:
    название отеля;
    широту и долготу отеля;
    страну и город, где расположен объект.

Запуск скрипта
Для запуска нужно ввести следующую команду в терминале, находясь в папке weather_alalysis:
python weather_app.py --data_path="data_dir" --output_path="out_dir"
data_dir - путь к архиву с данными
out_dir - путь к папке, в которой будут сохранены результаты (должна быть созданга заранее)
Пример:
python weather_app.py --data_path="data/hotels.zip" --output_path="D:/data"
*относительные пути поддерживаются

Примечание:
Проект использует медленный API для запроса геоданных. Поэтому для экономии времени рекомендуется
использовать кэшированные геоданные (data/cached_geocoding.zip) для предоставляемого с проектом
набора данных. Содержимое архива необходимо поместить в папку src
